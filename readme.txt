== Description ==

Das easySEO Plugin bietet eine einfache Moeglichkeit, grundlegende Optimierungen seines Blogs per Plugin
vorzunehmen. In der vorliegenden Version wird ein optimierter HTML Titel bestehend alleinig aus dem
Namen des jeweiligen Posts generiert. Des weiteren kann zu jedem Post eine Beschreibung fuer das HTML
Dokument mit dem Feld 'Auszug' erstellt werden.
Zudem wird eine XML-Sitemap sowie eine Urllist generiert, die sich automatisch bei jedem neuen Post
aktualisiert.

Dieses Plugin eignet sich fuer die schnelle und einfache Optimierung eines Wordpress Blogs. Dabei werden
in der vorliegenden Version noch keine administrativen Eingriffsmoeglichkeiten angeboten. Der Anspruch
bei diesem Plugin war und ist, schnell und einfach grundlegende Optimierungsarbeiten zu erledigen, ohne
sich bei der Installation eines Blogs damit lange aufhalten zu muessen.

Fuer dieses Plugin benoetigen Sie einen Webspace mit mindestens PHP 5.

== Installation ==

1. Uploaden Sie den Ordner `easyseo` in das `/wp-content/plugins/` Verzeichnis
2. Aktivieren Sie das Plugin im Wordpress Administrationsbereich
3. Achten Sie darauf, ob die Dateien sitemap.xml und urllist.txt vom Script erzeugt werden konnten.
   Sollte dies nicht der Fall sein, muessen diese manuell erstellt und per FTP auf Ihren
   Webspace hochgeladen werden. Aendern Sie die Dateiberechtigung fuer die Sitemap Dateien ggf. auf 777 (CHMOD).
